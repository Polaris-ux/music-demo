# music-demo
SpringBoot+Mybatis音乐网站
# 技术
## 前端
Vue <br/>
Vue-Router <br/>
ElementUI <br/>
VCharts <br/>

## 后端
SpringBoot <br/>
MyBatis <br/>

## 前端结构
```
├─api -> 通信，发起请求
├─assets
│  ├─css
│  ├─data
│  ├─img
│  │  └─swiper
│  └─js
├─components ->组件
│  └─search
├─mixins ->工具类
├─pages ->页面
├─router ->路由
└─store ->缓存
```

## 使用
### 后端代码
拉取到本地后，修改数据库以及redis的相关参数后即可直接运行
### 前端
需要修改api/下文件的请求地址
```
npm install
npm run dev
```
